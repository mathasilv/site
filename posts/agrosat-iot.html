AGROSAT-IOT

DATA: 2025-11-04 | LINGUAGEM: C++

Sistema de IoT para agricultura de precisão com comunicação via satélite. O projeto combina sensores de campo com transmissão de dados para monitoramento remoto de lavouras.

Desenvolvido como parte de um trabalho maior de pesquisa em telecomunicações para áreas rurais sem cobertura de internet convencional.

Características

  • Hardware: Baseado em ESP32 com módulos LoRa
  • Telemetria: Coleta dados de temperatura, umidade e outros sensores
  • Comunicação: Suporte a LoRa e protocolos satelitais
