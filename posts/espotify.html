ESPOTIFY

DATA: 2023-07-13 | LINGUAGEM: C++

Integração entre ESP32 e a API do Spotify. Controle sua música direto de um microcontrolador!

Esse projeto surgiu da vontade de criar um controle físico para o Spotify, usando botões e displays em um dispositivo embarcado.

Características

  • API: Conexão direta com a Web API do Spotify
  • Interface: Controles físicos para play, pause e skip
  • Display: Exibe informações da música atual
