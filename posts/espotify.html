ESPOTIFY - DISPLAY SPOTIFY PARA ESP8266/ESP32

DATA: 2023-07-13 | LINGUAGEM: C++ | PLATAFORMA: ESP8266/ESP32

Sistema embarcado para exibição em tempo real da música atualmente em reprodução no Spotify, utilizando microcontroladores ESP8266 ou ESP32 com display OLED.

Descrição

Este projeto permite visualizar informações da música que está sendo reproduzida na sua conta do Spotify em um display OLED 128x32. São exibidos o nome da faixa, artista e uma barra de progresso da reprodução.

Requisitos de Hardware

  • Microcontrolador: ESP8266 (NodeMCU v2) ou ESP32
  • Display: OLED SSD1306 128x32 pixels (interface I2C)
  • Cabos de conexão

Esquema de Conexão (I2C)

Display ESP8266
SDA D2 (GPIO 4)
SCL D1 (GPIO 5)
VCC 3.3V
GND GND

Bibliotecas Utilizadas

  • ESP8266WiFi / WiFi: Conexão com a rede
  • ESP8266HTTPClient / HTTPClient: Requisições HTTP
  • ArduinoJson: Parsing de JSON
  • Adafruit_SSD1306: Controle do display OLED
  • Base64: Codificação para autenticação

Configuração

  1. Criar aplicativo no Spotify Developer
  2. Configurar Client ID e Client Secret no código
  3. Obter Refresh Token via OAuth2
  4. Compilar e fazer upload para o ESP

Limitações

  • Requer conta Spotify Premium para algumas funcionalidades
  • Token de acesso expira a cada hora (renovado automaticamente)
  • Display pequeno limita quantidade de texto visível
