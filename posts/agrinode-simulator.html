AGRINODE-SIMULATOR

DATA: 2025-11-25 | LINGUAGEM: C++ | PLATAFORMA: ESP32

Simulador de nós agrícolas para testes e desenvolvimento de sistemas IoT voltados para agricultura. Este projeto complementa o AgroSat-IoT, permitindo emular múltiplos sensores de campo sem necessidade de hardware real.

Objetivo

Facilitar o desenvolvimento e debug de aplicações de agricultura de precisão, especialmente para o projeto AgroSat-IoT. O simulador gera dados realistas de sensores que podem ser transmitidos via LoRa para testes de integração.

Funcionalidades

  • Emulação de Múltiplos Nós: Simula vários sensores de campo simultaneamente
  • Dados Realistas: Gera leituras de temperatura, umidade do solo e ambiente
  • Comunicação LoRa: Transmite pacotes no mesmo formato do sistema real
  • Configuração Flexível: Parâmetros ajustáveis para diferentes cenários de teste
  • Logs Detalhados: Registro de todas as transmissões para análise

Casos de Uso

  • Testar a estação terrestre (NPA-GS) sem satélite real
  • Validar algoritmos de processamento de dados
  • Desenvolver novas funcionalidades do AgroSat-IoT
  • Demonstrações e apresentações do sistema

Hardware Suportado

  • Placa: ESP32 com módulo LoRa (SX1276/SX1278)
  • Antena: Helicoidal ou dipolo para 433/868/915 MHz

Integração com AgroSat-IoT

O simulador gera pacotes compatíveis com o protocolo do sistema AgroSat-IoT, incluindo:

  • ID do nó sensor
  • Timestamp da leitura
  • Dados de sensores (temperatura, umidade, etc.)
  • Checksum para integridade
